# Savoury Spoon

A modern React restaurant website for **Savoury Spoon** — featuring the full menu, allergen profiles, and a premium dark-themed UI.

## Features

- Responsive single-page design with smooth scroll navigation
- Interactive menu with category tabs
- Full allergen transparency (Egg, Milk, Gluten, Soy, Allergen Free)
- Animated sections powered by Framer Motion
- Built with React 19, TypeScript, Vite, and Tailwind CSS v4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Menu Categories

- **Egg Specialties** — 14 signature egg dishes
- **Snacks & Starters** — Nachos, fries, samosas & more
- **Sandwiches & Rolls** — Veg cheese sandwich, paneer roll, aloo paratha
- **Mains** — Schezwan Chowmein
- **Beverages** — Masala tea, British tea, Oreo shake
